# PostgreSQL Practice Project (First Principles)

This project is a **hands-on practice backend** built to understand **PostgreSQL from first principles**, without relying on ORMs initially.  
The goal is to learn how real backend systems interact with relational databases using **raw SQL, migrations, and proper configuration**.
